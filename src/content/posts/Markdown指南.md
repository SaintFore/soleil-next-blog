# Markdown基础语法指南

Markdown是一种轻量级标记语言，创建于2004年，现在已成为世界上最流行的标记语言之一。

## 基础语法

### 标题

使用`#`符号表示标题，一个`#`是一级标题，两个`#`是二级标题，以此类推。

### 强调

*斜体* 或 _斜体_

**粗体** 或 __粗体__
